# MaterialWiki
MediaWiki skin based on [Material Design Lite](https://github.com/google/material-design-lite)

## Installation

On MediaWiki directory:
```
$ cd skins

$ git clone https://github.com/brunolimawd/MaterialWiki.git

$ cd MaterialWiki

$ bower install
```
